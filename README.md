# Computational Methods

This repository contains Python implementations of numerical methods used in physics and scientific computing.

## Topics Covered

- Curve Fitting
- Bisection Method
- Newton-Raphson Method
- Histogram Analysis
- Time-correlated distributions
- Interpolation and Extrapolation
- Differentiation and Integration
- Time Integration Techniques
- Fourier Transform
